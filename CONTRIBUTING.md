# Contributing to Hashi Bridger

### Want to contribute to Hashi Bridger?
##### Thanks! Happy to have you here :) See below for details on what's in the works, currently, and what's on the roadmap

### Contributing etiquette
##### If you wish to contribute (sweet!!), fork the repo, clone it to your machine, create a [well-named branch](https://conventional-branch.github.io/), contribute as per any of the issues below, commit [with good messages](https://www.conventionalcommits.org/en/v1.0.0/#summary), push to your fork and open a pull request in this repo
